# Üretim Duruş Analizi (Örnek Verilerle)

Bu projede Excel kullanılarak örnek verilerle üretim hattı duruş analizi yapılmıştır.  
Pivot Table ve Dashboard yardımıyla duruş nedenleri ve hat bazlı performans analiz edilmiştir.  

## İçerik
- Manufacturing_Line_Productivity.xlsx → Ham veriler ve analiz
- Pivot Table → Duruş nedenleri, hat bazlı süreler
- Dashboard → Grafiklerle görselleştirilmiş analiz

## Not
Bu projedeki veriler **örnek amaçlıdır**, gerçek üretim verilerini temsil etmez.
